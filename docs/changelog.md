# OrbitAN Changelog

Semantic Versioning and release dates for OrbitAN.

## v0.1.0 - 2024-10-01
- Initial planet rendering with a static orbit and nebula backdrop.
- Core 4-layer canvas pipeline registration planned.
- Base structure for localStorage persistence scaffold.

## v0.2.0 - 2024-12-15
- Added Orbit Modes panel: first set of interactive panels in Orbit Mode sidebar.
- Implemented boundaries and basic navigation for Daily view.
- Introduced tasks data model skeleton and type mapping for Work/Study/Meeting/Personal.

## v0.3.0 - 2025-04-22
- WeekView with 7-column grid and 06:00–18:00 axis.
- Added rAF animation for planet hands and dynamic clock visuals.
- Unified localStorage storage approach (single namespace) for all data.
- Minor UI polish and performance improvements on rendering pipeline.

## Current
- Ongoing: Refinements to all six time-management methodologies, adding more robust data structures, and stabilizing persistence.
- Expect continued improvements to WeekView interactions and Orbit Mode panel persistence.
